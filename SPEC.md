# 婴儿声音翻译器 - 项目规范

## 1. 项目概述

- **项目名称**: 婴儿声音翻译器 (Baby Voice Translator)
- **项目类型**: 响应式 Web 应用 (手机端优先，适配 Web 端)
- **核心功能**: 采集婴幼儿声音，AI 识别并翻译成人类语言，支持文字转语音
- **目标用户**: 父母、祖父母等照顾婴幼儿的家庭成员

## 2. 功能需求

### 2.1 核心功能
- **一键录音**: 点击按钮开始录音，再次点击停止
- **音频上传**: 支持上传本地音频文件 (mp3, wav, m4a)
- **AI 翻译**: 将婴幼儿声音/哭声翻译成人类语言
- **多语言支持**: 中文、英语、日语、韩语等
- **语音播报**: 翻译结果可转为语音播放
- **历史记录**: 保存翻译历史

### 2.2 用户流程
1. 用户打开应用 → 首页
2. 选择"一键录音"或"上传音频"
3. 录音/上传完成后，显示识别结果
4. 选择目标语言
5. 查看翻译结果
6. 可选择语音播放或分享

### 2.3 页面结构
- 首页: 录音入口 + 上传入口 + 温馨提醒
- 录音页面: 录音动画 + 计时器 + 停止按钮
- 结果页面: 翻译结果 + 语言选择 + 语音播放
- 历史页面: 历史记录列表
- 设置页面: 语言偏好 + 关于

## 3. UI/UX 设计规范

### 3.1 色彩系统
```
主色调: #EE2B5B (柔和粉红)
背景色: #FCF8F8 (温暖白)
强调色:
  - 蓝色: #E0F2FE
  - 粉色: #FCE7F3
  - 黄色: #FEF3C7
文字色:
  - 主文字: #1E293B
  - 次文字: #64748B
```

### 3.2 字体
- 主字体: Plus Jakarta Sans
- 中文备选: Noto Sans SC

### 3.3 布局
- 移动端优先 (最大宽度: 430px)
- 底部导航栏
- 圆角卡片 + 柔和阴影

## 4. 技术栈

- **前端**: React 18 + Tailwind CSS + Vite
- **后端**: Node.js + Express
- **数据库**: SQLite (better-sqlite3)
- **AI**: Anthropic Claude API (可配置)
